# NN-Webview2
This is a webview repository for the Windows platform that supports embedding assets,
has no dependencies, and is based on C++20.

## Fast Start
```powershell
./build.sh
./build/a
```

## Why develop
It's 2026, and I'm fed up with apps built on the Electron framework.
Especially now, during this period of rapid AI development, 99% of AI applications use the Electron framework!!!
Some even use web frameworks to build CLIs!!! Okay, I accept it—what choice do I have?
But Electron bundles an entire browser just for cross-platform compatibility.
In contrast, I don't really care if an app is cross-platform or not, so frameworks based on webviews are clearly
better—they're lightweight and don't need to package a whole browser.
However, the webview frameworks available on the market overcomplicate things in the name of cross-platform support.
That's why I developed this webview project for Windows.
It has no dependencies and can be compiled and run directly.
Come on, what are you waiting for?
